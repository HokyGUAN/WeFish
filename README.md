# *WeFish User Guide*

<br>


> ### Compile WeFish
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

<br>

> ### Startup WeFish Server
```
$ cd build/server
$ ./server <Group Port>
```

<br>

> ### Startup WeFish Client
```
$ cd build/client
$ ./client <IP Address> <Group Port> <Your Chat Name>
```
